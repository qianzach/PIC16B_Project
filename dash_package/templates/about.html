About the Project

This project was inspired by our shared interest in watching Anime. After scraping IMDb data of various animes, we found that our data was networked data - many actors were interconnected with each other through common shows. This motivated us to create a movie/tv show recommendation engine based on the raw networked data of actors and projects they are a part of.

The recommendation engine combines two essential elements: a Scrapy web scraper and an interactive Dash app to vizualize the recommendations! We include the following media:
